## Description

> ! This is a very early proof of concept

CSSH is a graphical ssh helper written in C++ using [ncurses](https://invisible-island.net/ncurses/).

## Build

You can build cssh using cmake

```sh
mkdir build
cd build

cmake ..
make

./cssh
```
